# Core 子模块 — 已废弃（实验性版本）

> **DEPRECATED / 已废弃**
>
> 本目录为实验性新架构版本，仅供技术探索使用，**请勿在生产环境中引用**。

## 说明

`core/` 目录是 shared 库早期尝试的分层架构设计，目标是将核心能力（配置、日志、数据库等）独立为一层，与客户端层分离。

由于该架构尚未经过充分验证，且根目录的稳定版已被 M8 等大量模块广泛使用，为避免破坏现有功能，**当前版本保留根目录实现作为主版本**。

## 推荐使用

请使用 `shared/` 根目录下的稳定版模块：

| 功能 | 稳定版位置 | 说明 |
|------|-----------|------|
| 全局配置 | `shared/config.py` | 生产环境在用 |
| 统一日志 | `shared/logger.py` | 生产环境在用 |
| 模块调用 | `shared/module_client.py` | 生产环境在用 |
| 进程管理 | `shared/process_manager.py` | 生产环境在用 |
| LLM 客户端 | `shared/llm_client.py` | 生产环境在用 |
| 语音引擎 | `shared/voice_engine.py` | 生产环境在用 |

## 未来规划

未来版本可能会将 `core/` 和 `clients/` 中的优秀设计整合进主版本，但**不保证接口兼容性**。如需使用本目录中的代码，请自行评估风险。

## 当前状态

- 状态：实验性 / 不推荐生产使用
- 维护：低优先级
- 兼容性：不保证与根目录版本一致
