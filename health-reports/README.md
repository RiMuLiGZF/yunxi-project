# 云汐系统体检报告库

> 本目录存放云汐系统各模块的体检报告和技术文档。

---

## 文档编号规则

```
YX-{类型}-{编号}-{名称}-{版本}.md
```

**类型**：
- `SYS` — 系统级报告（总览、架构、路线图等）
- `MOD` — 模块级报告（单个模块体检）
- `MAN` — 管理类文档（主理人手册、运维手册等）

**编号**：
- 系统级：00 开始
- 模块级：00-11 对应 shared + M1-M11
- 管理类：00 开始

## 报告说明

每份模块体检报告包含以下章节：

1. **基本信息** — 版本、端口、代码规模等元数据
2. **功能实现情况** — 按核心/重要/一般分级的功能清单
3. **系统定位与作用** — 模块在系统中的角色和上下游关系
4. **架构设计** — 目录结构、技术栈、核心组件、数据流
5. **代码质量评估** — 规范性、可维护性、测试覆盖、依赖管理
6. **存在的问题** — 按严重/重要/一般分级的问题清单
7. **优化建议** — 按高/中/低优先级的改进建议
8. **健康评分** — 综合评分 + 五维分项评分
