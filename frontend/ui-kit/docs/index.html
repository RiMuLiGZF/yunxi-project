快速开始

引入组件库样式和脚本即可使用，支持完整引入和按需引入两种方式。

完整引入

最简单的方式，一次性引入所有组件样式和脚本。 

<link rel="stylesheet" href="ui-kit/src/index.css">
<script src="ui-kit/src/index.js"></script>
<script src="ui-kit/src/utils/index.js"></script>
按需引入

只引入需要的组件，减少体积。需先引入基础样式。 

<!-- 基础样式（必须） -->
<link rel="stylesheet" href="ui-kit/src/styles/variables.css">
<link rel="stylesheet" href="ui-kit/src/styles/reset.css">
<link rel="stylesheet" href="ui-kit/src/styles/base.css">

<!-- 按需引入组件 -->
<link rel="stylesheet" href="ui-kit/src/components/button/button.css">
<script src="ui-kit/src/components/button/button.js"></script>
纯 CSS 使用

也可以只使用 CSS 类名，不使用 JS API。 

<button class="yx-btn yx-btn--primary">主按钮</button>
<input class="yx-input" placeholder="请输入">
<div class="yx-card">...</div>

Input 输入框

基础文本输入组件，支持多种类型和验证状态。

<input class="yx-input" placeholder="请输入内容">
<input class="yx-input yx-input--sm" placeholder="小号">
<input class="yx-input yx-input--lg" placeholder="大号">
// JS API
const input = YunXiUI.Input.create({
  placeholder: '请输入邮箱',
  label: '邮箱地址',
  required: true,
  helpText: '我们不会泄露您的邮箱',
  onChange: function(e) { console.log(e.target.value); }
});

// 设置验证状态
YunXiUI.Input.setStatus(input, 'error', '邮箱格式不正确');

Utils 工具函数

内置常用工具函数库，包含类型判断、DOM操作、日期格式化等。

Type 类型判断
YunXiUI.Utils.Type.isString('hello') // true
YunXiUI.Utils.Type.isArray([1, 2]) // true
YunXiUI.Utils.Type.isEmpty('') // true
YunXiUI.Utils.Type.isDate(new Date()) // true
Date 日期格式化
format(new Date(), 'YYYY-MM-DD HH:mm:ss')
// "2024-01-15 14:30:00"

relativeTime(new Date(Date.now() - 300000))
// "5 分钟前"
Number 数字格式化
format(1234567) // "1,234,567"
currency(99.9, '¥') // "¥99.90"
formatBytes(1048576) // "1 MB"
percent(0.85, 1) // "85.0%"
