## 开发原则

1. **数据安全第一**：所有用户数据仅本地存储，绝不上传
2. **最小权限**：默认最低权限，按需授权
3. **全链路审计**：所有操作可追溯
4. **零知识设计**：代码中不硬编码任何真实数据
5. **可验证安全**：安全特性可通过测试验证
