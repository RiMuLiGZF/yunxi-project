# M5 潮汐记忆系统 - 变更日志

## [2.4.0-REV2] - 2024-xx-xx

### 新增
- 新增四级密级标记系统（PUBLIC/INTERNAL/CONFIDENTIAL/TOP_SECRET）
- 新增三级域权限隔离（private/shared/core）
- 新增M8标准接口适配层
- 新增端云同步桥接（content_hash + sync_version）
- 新增数据脱敏模块
- 新增审计日志系统

### 优化
- 优化EI情绪推断引擎，支持效价-唤醒度模型
- 优化睡眠巩固算法，提升记忆迁移准确率
- 优化向量检索性能
- 优化配置管理，支持环境变量注入

### 安全
- TOP_SECRET级别数据强制AES-256-GCM加密
- 高涉密数据禁止云端同步
- 所有访问操作增加审计记录
- 对外接口默认数据脱敏
