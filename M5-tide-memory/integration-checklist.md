# M5 潮汐记忆系统 - 整合前检查清单

> ⚠️ 整合前必须逐项确认，确保系统安全和数据完整

## P0 - 安全检查（必须通过）

- [ ] **数据零泄露**：确认没有任何真实用户记忆数据被打包
- [ ] **数据库文件**：确认没有 .db / .sqlite / .json 数据文件被打包
- [ ] **密钥安全**：配置文件中的密钥是示例值，不是真实密钥
- [ ] **加密算法**：TOP_SECRET级数据使用AES-256-GCM加密
- [ ] **本地存储**：高涉密数据仅本地存储，不同步云端
- [ ] **审计日志**：所有读写操作有审计记录
- [ ] **数据脱敏**：对外接口返回内容已脱敏
- [ ] **权限隔离**：三级域权限正常工作

## P1 - 功能检查（建议通过）

- [ ] **四层模型完整**：L0沙滩 / L1浅水 / L2深水 / L3深海
- [ ] **四级密级标记**：PUBLIC / INTERNAL / CONFIDENTIAL / TOP_SECRET
- [ ] **EI情绪推断**：效价-唤醒度模型正常
- [ ] **睡眠巩固**：记忆迁移、语义蒸馏正常
- [ ] **M8接口**：标准错误码和响应格式
- [ ] **端云同步**：content_hash + sync_version机制
- [ ] **测试覆盖**：核心功能测试通过
- [ ] **文档完整**：README + API文档齐全

## P2 - 性能优化（可选）

- [ ] **向量检索性能**：P95 < 100ms
- [ ] **并发处理**：支持100+ QPS
- [ ] **内存占用**：稳态 < 500MB
- [ ] **启动时间**：冷启动 < 5s
