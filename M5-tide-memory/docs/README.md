# M5 潮汐分层记忆系统

> ⚠️ **高涉密模块** - 所有用户记忆数据仅本地加密存储，绝不上传云端

## 概述

M5 潮汐分层记忆系统是云汐项目的核心记忆模块，采用四层潮汐模型模拟人类记忆的形成、巩固和遗忘过程。

## 四层潮汐模型

| 层级 | 名称 | 保留时间 | 容量 | 存储方式 | 加密 |
|------|------|----------|------|----------|------|
| L0 | 沙滩层 | ~1小时 | 100条 | 内存 | - |
| L1 | 浅水层 | ~1天 | 1,000条 | SQLite | 可选 |
| L2 | 深水层 | ~30天 | 10,000条 | SQLite+向量 | 可选 |
| L3 | 深海层 | 永久 | 100,000+ | 文件系统 | AES-256-GCM |

## 核心特性

- **三级域权限隔离**：Agent私有域 / 协作共享域 / 全局核心域
- **四级密级标记**：PUBLIC / INTERNAL / CONFIDENTIAL / TOP_SECRET
- **EI情绪推断**：基于效价-唤醒度模型
- **睡眠记忆巩固**：压缩 / 合并 / 索引重建
- **审计日志系统**：每次读写都有记录
- **M8标准接口**：统一错误码和响应格式

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 环境检查
python src/main.py --check

# 手动触发巩固
python src/main.py --consolidate
```

## 安全说明

- 所有用户记忆数据**仅本地存储**，绝不上传云端
- TOP_SECRET 级别数据使用 **AES-256-GCM** 加密
- 所有访问操作均有**审计日志**记录
- 对外接口默认开启**数据脱敏**
- 配置文件中的密钥使用**环境变量**注入
