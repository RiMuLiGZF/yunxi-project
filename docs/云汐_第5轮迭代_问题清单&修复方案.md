4. 静态资源未做 gzip 压缩

## 三、定制修改方案

### 优化 1：缓存层引入

**方案**：
- 使用内存缓存（LRU）缓存热点查询结果
- 缓存过期时间根据数据类型设置
- 提供缓存统计和手动清理接口

### 优化 2：数据库索引优化

**方案**：
- 为常用查询字段添加索引
- 优化慢查询 SQL 语句
- 添加查询性能监控

### 优化 3：静态资源压缩

**方案**：
- FastAPI 启用 GZipMiddleware
- 压缩 HTML/CSS/JS 静态资源
- 设置合理的缓存控制头
