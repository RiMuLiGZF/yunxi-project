## 一、系统背景

云汐是一个模块化、可扩展的个人 AI 操作系统。它以"场景化智能陪伴"为核心设计理念，围绕用户的日常生活、学习、工作、社交、情感等多维需求，构建了一套从底层算力调度到上层业务场景的完整技术栈。

系统采用微服务架构，每个模块（M0-M12）独立部署、通过 HTTP/MCP 协议通信，由 M8 控制塔统一纳管。核心特色包括：

- **潮汐分层记忆**：模拟人类记忆的形成、巩固和遗忘，四级分层（沙滩→浅水→深水→深海）
- **多场景智能切换**：根据用户行为和上下文，自动在工作/学习/生活/社交/情感等场景间暖切换
- **端云协同调度**：本地隐私优先，云端算力增强，智能分配计算任务
- **可插拔技能生态**：18+ 内置技能 + MCP/A2A 协议扩展，能力无限扩展
- **GPU 加速支持**：向量检索、本地推理全面支持 CUDA + TensorRT 加速

## 三、系统架构总览

```
┌─────────────────────────────────────────────────────┐
│                    M0 主理人管控台                    │  最高权限管控（Owner Only）
├─────────────────────────────────────────────────────┤
│                    M8 管理控制塔                      │  统一入口/认证/纳管/算力调度
├──────────┬──────────┬──────────┬──────────┬─────────┤
│   M1     │   M2     │   M7     │   M9     │  M12    │  能力层
│ Agent集群 │ 技能集群  │ 工作流编排 │ 开发工坊  │ 安全盾  │
├──────────┼──────────┼──────────┼──────────┼─────────┤
│   M4     │   M5     │  M11     │          │         │  业务/数据层
│ 场景引擎  │ 潮汐记忆  │ MCP总线  │          │         │
├──────────┼──────────┼──────────┼──────────┼─────────┤
│   M3     │   M6     │  M10     │          │         │  基础设施层
│ 端云协同  │ 硬件外设  │ 系统卫士  │          │         │
└──────────┴──────────┴──────────┴──────────┴─────────┘
         shared/  models/  config/  scripts/           ← 公共基础
```

### 层级说明

1. **管控层（M0+M8）**：系统入口和管理中枢，负责认证、权限、监控、算力调度
2. **能力层（M1/M2/M7/M9/M12）**：提供核心能力支撑，Agent调度、技能扩展、工作流、开发工具、安全防护
3. **业务/数据层（M4/M5/M11）**：业务场景调度、记忆系统、MCP服务总线
4. **基础设施层（M3/M6/M10）**：端云协同、硬件接入、系统监控守护
