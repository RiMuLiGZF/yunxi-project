### 1.3 当前主要问题与挑战

1. **架构层面**
   - M8路径大小写不一致（已发现，待修复）
   - 各模块间调用链路较长，缺少统一的服务治理
   - 分布式能力处于雏形，缺少完整的集群管理

2. **能力层面**
   - 新能力（画像/提醒/多模态）尚未与现有场景深度融合
   - 个性化能力停留在基础层，未形成闭环学习
   - 多模态理解仅有接口，缺少本地模型支撑

3. **体验层面**
   - 前端各场景页面分散，缺少统一的交互范式
   - 缺少端到端的用户体验链路设计
   - 语音交互不够自然（缺少打断、纠错等）

## 二、总体战略方向

### 2.1 愿景

**打造一个"懂你、陪你、帮你"的全场景AI伙伴**

- **懂你**：深度理解用户偏好、习惯、情绪
- **陪你**：全天候陪伴，主动关心，情感支持
- **帮你**：高效完成任务，学习成长，生活管理

### 2.2 核心战略：三大支柱

```
                    ┌─────────────────┐
                    │   个性化深度    │
                    │  Personalization│
                    └────────┬────────┘
                             │
              ┌──────────────┼──────────────┐
              │              │              │
    ┌─────────▼───┐  ┌──────▼──────┐  ┌───▼─────────┐
    │ 主动智能     │  │   云汐大脑   │  │   全场景覆盖  │
    │ Proactive   │  │   Brain     │  │  Omni-chan  │
    └─────────────┘  └─────────────┘  └─────────────┘
```

**支柱一：云汐大脑（认知核心）**
- 三层模型调度深化（3B调度 + 7B执行 + 云端增强）
- 长期记忆与知识图谱
- 推理与规划能力
- 多模态融合理解

**支柱二：个性化深度（懂你的核心）**
- 用户画像从"静态标签"到"动态人格"
- 闭环学习：交互 → 理解 → 适应 → 验证
- 情感计算与共情能力
- 个人知识库构建

**支柱三：主动智能（从被动到主动）**
- 情景感知：时间/地点/设备/状态/社交
- 主动提醒与建议
- 预测性服务
- 习惯养成与行为引导
